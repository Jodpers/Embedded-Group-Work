void update_display ( void   ) 

Display State Machine.

This function scrolls the characters from the display_buffer across the 7 seg display. This approach means another thread can send the string to be displayed to display_buffer and then continue what it was doing without being blocked. While this thread handles the scrolling.

The menu_set variable is used to determine if the display is in menu mode. If so, the string should display its first 4 chars after its finished scrolling, to give a persistent indication of which choice is currently selected.

Parameters:
Void. 
Returns:
Void.
