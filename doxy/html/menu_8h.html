void menu_select ( void   ) 

Menu selection routine.

This function is called by pressing ENTER_MENU (button E) while state machine is in 'WAITING_LOGGED_IN' state and no track number has been input.

Options are navigated using the FORWARD (F) and BACK (B) buttons or pressing the relevant option number on the keypad.

Options are selected by pressing ENTER_MENU (E) or ACCEPT (A).

Void.

void volume ( void   ) 

Volume control from the Menu.

This function uses the Alsa API to get and set the volume for the master channel. The use inputs a two digit number to set the volume, or presses the 'F' and 'B' buttons to adjust the value incrementally.

The volume control behaves differently from the menu, so menu flag is set to FALSE.
